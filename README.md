# 🔗 Blockchain Assignments Collection

A comprehensive collection of blockchain-related assignments, ranging from simple smart contracts to decentralized applications (DApps) and DAO implementations.

## 📁 Repository Structure

Each directory contains a specific assignment with its own source code, documentation, and execution screenshots.

| Assignment | Title | Description | Key Tech |
|------------|-------|-------------|----------|
| [**ass1**](./ass1/) | MessageStore Smart Contract | A simple Solidity contract to store and update a message on the blockchain. | Solidity, Remix, MetaMask |
| [**ass2**](./ass2/) | Smart Contract Implementation | Further exploration of smart contract functionalities and interactions. | Solidity, Sepolia |
| [**ass3**](./ass3/) | Blockchain Frontend Integration | Connecting a web frontend to a smart contract using `ethers.js`. | HTML/CSS, JS, Ethers.js |
| [**ass4**](./ass4/) | DApp Development | Building a functional Decentralized Application with interactive components. | Web3.js/Ethers.js, Solidity |
| [**ass5**](./ass5/) | DAO Implementation | Implementing a Decentralized Autonomous Organization (DAO) logic. | Governance, Solidity |

---

## 🛠️ General Tech Stack

- **Smart Contracts:** Solidity (0.8.x)
- **Development Environment:** Remix IDE
- **Wallet/Provider:** MetaMask
- **Libraries:** Ethers.js / Web3.js
- **Frontend:** HTML5, CSS3, JavaScript
- **Network:** Sepolia Testnet

---

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Strix33/blockchainass.git
   ```
2. **Navigate to an assignment:**
   ```bash
   cd ass1
   ```
3. **Follow the instructions** in each assignment's specific `README.md` to run or deploy the code.
